## I. Central EU Funding Portals

Primary entry points for accessing European grants and public-private funding opportunities.

| Organization | URL | Purpose |
|-------------|-----|---------|
| **EU Funding & Tenders Portal (SEDIA)** | https://ec.europa.eu/info/funding-tenders/opportunities/portal/ | Central entry point for all centrally-managed EU funding |
| **EUFundingPortal.eu** | https://eufundingportal.eu | Private aggregator with searchable database |
| **EuroAccess Database** | https://euro-access.eu/en/calls | Multi-program database with advanced filtering |

### Key Features
- Partner search functionality
- Database of past and ongoing EU-funded projects
- News, webinars, and training materials
- Multi-language support across EU languages
